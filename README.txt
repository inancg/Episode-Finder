Episode Finder(R) Version 1.0 30/12/2014

Notes About Software

Setting up can take up to a minute with respect to your computer performance.
Process can be tracked from the Command Line

Setting Up

Run EpisodeDataGathering.py (Python 3 is recommended).
Do this only once.

Usage

Run SearchEpisode.py
Search keywords.
The results will show ordered from most frequent to less.

Troubleshooting

If there are any mistakes with the setup;
  Run Reset.py
  Setup again.
